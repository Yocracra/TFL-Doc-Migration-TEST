## 1\. Purpose of This Safety Manual

This document outlines the minimum safety information required to operate the NeoDen FP 2636 in The Fab Lab. It focuses on:

  * Hazards that can cause injury or damage
  * Required protective measures
  * Emergency response procedures



This document does not teach you how to operate the machine. Refer to the linked Operations Manual for usage instructions.

## 2\. Who May Use This Machine

This machine may be used by:

  * All Fab Lab users



Usage conditions:

  * Staff Presence
  * Safety Manual Acknowledgement
  * Operations Manual Acknowledgement
  * Solder Paste is Lead Free



## 

## 3\. Primary Hazards

Be aware of the following hazards when using this machine:

  * Pinch Points
  * Hazardous chemicals


  * Note that solder paste containing lead is not permitted for use



## 4\. Required Personal Protective Equipment (PPE)

The following PPE is required when operating this machine:

  * Nitrile Gloves
  * No loose fitted clothing or jewelry



## 5\. General Safety Rules

Always follow these rules when using this machine:

  * Keep hands, hair, clothing, and loose items clear of moving parts and solder paste
  * Do not operate the machine if you are unsure how to proceed



## 6\. Incident & Injury Response

If an incident occurs:

  1. Stop the machine use
  2. Move to a safe distance
  3. Notify Fab Lab staff immediately
  4. Rinse any solder paste from hands and body thoroughly
  5.
