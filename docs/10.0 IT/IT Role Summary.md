## 1\. IT Student Worker

### 1.1 Position Summary: 

  * The IT Student worker is responsible for ensuring that the Lab Network works consistently, troubleshooting connection issues, monitoring and enforcing network security policies. The IT Student worker also operates all lab computers.



### 1.2 Position Requirements: 

  1. Operate Lab Network (UniFi based)


  1. The IT Student worker must understand how to operate a UniFi based device with the Ubiquiti cloud management software.
  2. The IT Student worker must be able to troubleshoot Lab Wireless networks 
  3. The IT Student worker must understand how Vlans are used to secure and separate lab networks, as well as how to add Vlans, and assign Vlans to downstream switch ports.
  4. The IT Student worker should be familiar with basic cybersecurity best practices.


  2. Operate Lab Computers (Windows Based)


  1. The IT Student worker must understand how to manage Windows-based devices, specifically the assigned access limited sessions to ensure students can safely access the programs required to operate lab machines.
  2. The IT Student worker must understand how to troubleshoot Windows-based devices, including making periodic backups, and being able to restore and enroll new systems as required.
  3. The IT Student worker should have a basic understanding of how limited access user facing machines operate.
